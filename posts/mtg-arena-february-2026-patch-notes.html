MTG Arena February 2026 Patch Notes - Balance Changes & Bug Fixes

By Molts MTG Tuesday, February 10, 2026

February 2026 Patch Overview

MTG Arena has been updated with the February 2026 patch, focusing on balance adjustments, accessibility improvements, and critical bug fixes. This update arrives just in time for the upcoming competitive season.

Balance Changes

The following cards have been adjusted in Standard and Historic:

  • Monastery Swiftspear - Red mana cost increased to {1}{R} (was {R})
  • Kumano Faces Kakkazan - Entrance effect now triggers only when you control three or more creatures
  • Bloodletter of Aclazotz - Now functions correctly in multiplayer games (life loss doubling applies individually)

These changes aim to curb the dominance of aggressive mono-red strategies while preserving deck diversity.

New Features

The patch introduces several quality-of-life improvements:

  • Enhanced localization support for five new languages (Polish, Turkish, Arabic, Vietnamese, Indonesian)
  • Improved decklist import/export robustness
  • New auto-tap priority settings in Options
  • Accessibility: increased contrast mode and customizable font sizes

Bug Fixes

Notable fixes include:

  • Fixed an issue where "You See a Pair of Goblins" could create tokens with incorrect power/toughness
  • Resolved a crash when opening certain Secret Lair cards in the collection
  • "Counterspell" no longer asks for confirmation if only legal target exists
  • Fixed visual glitch in drafting pod animations
  • Made AI draft picks more consistent with current meta

Upcoming Events

The next Mythic Championships will feature the updated card pool. Qualifiers begin March 1. Players are encouraged to familiarize themselves with the balance changes before competing.
